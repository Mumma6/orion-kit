## What's Included

<CardGrid stagger>
  <Card title="🔐 Authentication" icon="lock">
    Custom JWT authentication with protected routes, middleware, and
    localStorage token storage for cross-origin compatibility.
  </Card>
  <Card title="🗄️ Database" icon="setting">
    Type-safe Drizzle ORM with Neon Postgres, automatic migrations, and Studio
    GUI.
  </Card>
  <Card title="✅ Validation" icon="approve-check">
    Zod schemas on frontend and backend with automatic type inference from
    Drizzle.
  </Card>
  <Card title="🎨 UI Components" icon="star">
    Shadcn/ui components with Radix UI primitives and Tailwind CSS styling.
  </Card>
  <Card title="📊 Analytics & Logging" icon="laptop">
    PostHog for product analytics, Axiom for structured logging, and Vercel
    Analytics.
  </Card>
  <Card title="🚀 Data Fetching" icon="rocket">
    TanStack Query with automatic caching, optimistic updates, and loading
    states.
  </Card>
  <Card title="📝 Forms" icon="document">
    React Hook Form with Zod resolvers for type-safe, performant form handling.
  </Card>
  <Card title="⚡ Background Jobs" icon="sun">
    Trigger.dev for scheduled tasks, async processing, and reliable job
    execution.
  </Card>
  <Card title="🧪 Testing" icon="magnifier">
    Vitest for fast unit testing with comprehensive test suite and coverage
    reports.
  </Card>
  <Card title="📦 Shared Types" icon="puzzle">
    Centralized TypeScript types package for consistency across entire monorepo.
  </Card>
  <Card title="🏗️ Monorepo" icon="seti:config">
    Turborepo for optimized builds with pnpm workspaces and shared packages.
  </Card>
  <Card title="📚 Documentation" icon="open-book">
    Astro Starlight docs site with search, guides, and comprehensive examples.
  </Card>
</CardGrid>
